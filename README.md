# Void Notes

A minimalist, brutalist-styled Markdown **second-brain** notepad built with Electron, React, and CodeMirror 6.

## Project Status

As of version 0.4.1, Void Notes is now focused exclusively on the minimalistic experience. The expanded version with plugin support has been moved to a separate project to maintain the simplicity and performance of the core application.

The expanded version will be available soon as a separate application for users who need advanced features like plugins and the marketplace.

## Download

Get the latest installer from the [Releases](https://github.com/PixelCodeGH/VoidNotes/releases) page.

## Development

For contributors only:

```bash
npm install
npm run dev    # Start Vite dev server + Electron (hot reload)
npm run dist:win   # Build Windows .exe installer
```

Setup scripts (`setup.sh` / `setup.bat`) are provided for environment preparation but are not required for regular use.
